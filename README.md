# 供应商门户系统 (Supplier Portal)

## 项目说明

这是一个基于 Vue 3 的供应商门户系统前端原型，包含完整的用户界面和交互功能。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Tailwind CSS** - 实用优先的 CSS 框架
- **纯 HTML/JS** - 无需构建工具，直接在浏览器运行

## 功能模块

### 1. 用户账户管理
- ✅ 登录页（密码登录 / 验证码登录）
- ✅ 忘记密码
- ✅ 首次登录账号设置
- ✅ 修改密码

### 2. 报价管理
- ✅ 上传报价单（Excel批量导入 / 手动录入 / 快速报价）
- ✅ 报价单查询（状态筛选、批次管理、行内编辑）

### 3. POD处罚管理
- ✅ 统计卡片（错误单量、预扣罚金额）
- ✅ 数据表格（17字段，横向滚动）
- ✅ POD照片预览（缩略图 + 大图查看）
- ✅ 申诉功能（4种申诉类型、附件上传）

### 4. 页面布局
- ✅ 顶部导航栏（Logo、主导航、语言切换、用户菜单）
- ✅ 左侧侧边栏（二级菜单结构）
- ✅ 响应式设计

## 快速开始

### 方式一：直接打开（推荐）

1. 下载项目文件
2. 直接用浏览器打开 `index.html` 文件
3. 无需安装任何依赖

### 方式二：本地服务器

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve
```

然后访问 `http://localhost:8000`

## 功能演示

### 登录
- 默认显示密码登录和验证码登录两种方式
- 验证码登录支持邮箱和短信两种方式
- 60秒倒计时功能
- 隐私政策勾选验证

### 报价管理
- 三种导入模式切换
- 批次日期设置（生效/失效）
- 预览确认弹窗
- 草稿保存功能

### POD处罚
- 实时统计数据展示
- 多条件筛选
- 照片缩略图 + 大图预览
- 申诉表单验证
- 附件上传管理

## 注意事项

1. 本项目为前端原型，数据为模拟数据
2. 所有交互功能已实现，但未连接后端 API
3. 图片使用占位符，实际使用时需替换真实图片
4. 建议使用现代浏览器以获得最佳体验

## 后续开发建议

如需转换为生产环境项目，建议：

1. 使用 Vue CLI 或 Vite 构建工具
2. 集成状态管理（Vuex / Pinia）
3. 连接后端 API
4. 添加表单验证库（如 VeeValidate）
5. 集成 Excel 处理库（如 xlsx）
6. 添加国际化支持（vue-i18n）
7. 实现路由管理（vue-router）
